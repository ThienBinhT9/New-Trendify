1. Các pattern prompt

A. Prompt để debug
Tôi đang bị lỗi:
[error]

Context:
[code]

Tôi đã thử:
[list]

Bạn hãy:

- tìm nguyên nhân
- fix trực tiếp vào code
- giải thích ngắn gọn

B. Prompt để design system
Tôi đang xây dựng:
[mô tả hệ thống]

Yêu cầu:
[list]

Bạn hãy:

- đề xuất architecture
- giải thích trade-off
- đưa ra flow

C. Prompt để tối ưu code
Giải thích cho tôi:
[topic]

Yêu cầu:

- ví dụ thực tế
- so sánh với cách khác
- khi nào nên dùng / không nên dùng
